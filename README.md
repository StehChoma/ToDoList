# ToDoList
Aplicativo de lista de tarefas, desenvolvido com React-Native
<br/>
Neste app você pode:
<br/> - Adicionar tarefas;
<br/> - Dar um check nas tarefas concluídas;
<br/> - Excluir tarefas;
<br/> - Acompanhar as quantidades de concluídas e criadas.
